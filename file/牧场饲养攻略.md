# 牧场饲养攻略
## 前提
- 所有动物每2天减少部分饱食度，可使用饲料喂养增加饱食度。当饱食度+清洁度<180时，动物将无法繁殖；当饱食度+清洁度＜170时，动物产出将显著减少。
- 所有种类的动物，均可通过带出游玩、好友拜访的方式增加剩余寿命，可增加最多3天寿命。
- 牧场饲养的动物在出售后可立即养殖新的宝宝窝，缩短动物养殖周期。
- 动物两个产出周期内饱食度维持100，第二个周期产出日额外获得一个奖励产出。
